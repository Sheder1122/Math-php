# 📐 Math Learning System (PHP/MySQL)

Учебное веб-приложение для решения и проверки математических задач по алгебре, геометрии и математическому анализу.  
Пользователи могут проходить задания, а система сохраняет результаты и формирует статистику успеваемости.

## 🚀 Функционал

### Управление пользователями
- Добавление нового пользователя
- Редактирование ФИО пользователя
- Удаление пользователя
- Просмотр списка всех пользователей

### Задания по алгебре
- **Квадратные уравнения** – ввод коэффициентов, проверка корней (включая комплексные)
- **Определитель 3×3** – вычисление по правилу треугольников
- **Метод Крамера** – решение системы трёх линейных уравнений
- **Сумма матриц 3×3** – поэлементное сложение
- **Вычисление sin(x)/x** – разложение в ряд Тейлора
- **Экспонента eˣ** – разложение в ряд
- **Неопределённые интегралы** – проверка введённой первообразной численным дифференцированием

### Геометрические задачи
- **Нахождение углов треугольника** по трём сторонам
- **Вычисление площади и высот** треугольника
- **Вычисление медиан, высот и биссектрис** треугольника
- **Определение типа треугольника** (прямоугольный, остроугольный, тупоугольный) по координатам вершин

## 🛠 Технологии
- **Backend:** PHP 7+
- **Database:** MySQL
- **Frontend:** HTML
- **Сервер:** Apache / XAMPP / OpenServer

## 📁 Структура проекта
```
├── connect.php         # Подключение к БД
├── users.php           # Вывод списка пользователей
├── add_user.php        # Форма добавления пользователя
├── add_user2.php       # Обработчик добавления
├── delete_users.php    # Форма удаления (чекбоксы)
├── delete_users2.php   # Обработчик удаления
├── updateuser.php      # Форма редактирования
├── updateuser2.php     # Обработчик обновления
├── stats.php           # Страница статистики (с агрегирующим JOIN)
├── index.html          # Главное меню
│
├── Algebra:
│   ├── kv_ur.php / kv_ur2.php         # Квадратное уравнение
│   ├── det.php / det2.php             # Определитель 3×3
│   ├── kramer.php / kramer2.php       # Метод Крамера

│   ├── sin.php / sin2.php             # sin(x)/x

│ 
├── Geometry:

│   ├── square.php / square2.php       # Площадь и высоты
│   ├── treug_mbh.php / treug_mbh2.php # Медианы/высоты/биссектрисы

│   └── integral.php / integral2.php   # Интегралы (проверка производной)
```
